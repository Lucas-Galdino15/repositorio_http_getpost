COMO FUNCIONA O GET?
O GET é um método HTTP utilizado para solicitar e buscar informações de um servidor, sem causar nenhuma alteração nos dados, sendo muito usado para consultas, como visualizar usuários ou acessar páginas.

COMO FUNCIONA O POST?
O POST é um método HTTP usado para enviar dados ao servidor, geralmente com o objetivo de criar ou cadastrar novas informações, como enviar um formulário ou adicionar um novo usuário em um sistema.
